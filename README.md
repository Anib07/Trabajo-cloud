# Proyecto Web - Venta de Balanceados

## Descripción

Este proyecto consiste en el desarrollo de una página web orientada a la venta y promoción de alimentos balanceados para animales. El sitio web permite a los usuarios visualizar un catálogo de productos, acceder a información relacionada con los animales para los cuales están destinados los alimentos y observar galerías de imágenes de diferentes especies.

El objetivo principal es ofrecer una plataforma informativa y visual que facilite la presentación de los productos disponibles y brinde información de interés para los clientes.

## Funcionalidades

* Menú principal de navegación.
* Tarjetas (Cards) para acceder a cada sección del sitio.
* Catálogo de productos balanceados.
* Información sobre diferentes animales.
* Galerías de imágenes de animales.
* Diseño organizado para facilitar la navegación del usuario.

## Tareas Realizadas

* Creación del menú principal de navegación.
* Implementación de tarjetas de acceso a cada galería.
* Desarrollo de la Galería 1 correspondiente al catálogo de productos.
* Desarrollo de la Galería 2 con información de los animales.
* Desarrollo de las Galerías 3 y 4 destinadas a la exposición de imágenes de animales.
* Organización de la estructura general del sitio web.

## Ejecución del Proyecto

1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar el archivo `index.html` en un navegador web.
4. También puede utilizar la extensión **Live Server** de Visual Studio Code para visualizar el sitio de forma local.

## 🚀 Ejecución con Docker

Para construir las imágenes y levantar los contenedores en segundo plano, ejecuta:

docker compose up -d --build

Pero, en todo caso, se dejo un deploy.bat para poder trabajar con git+docker a la vez.
